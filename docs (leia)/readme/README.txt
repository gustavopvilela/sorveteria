Projeto: Sorveteria

Equipe:
- Eric Felipe Santos;
- Gustavo Henrique Pereira Vilela.

Especificações:
O programa descreve das seguintes funcionalidades:
1. O cliente pode visualizar a página inicial, a de cardápio, a de contato e a página de fazer pedido;

2. O cliente pode fazer um pedido na sorveteria;
    2.1. Nesse pedido, ele pode visualizar quais itens colocar no carrinho, seus preços, a quantidade e o preço total;
    2.2. O cliente também pode excluir ou alterar a quantidade de itens no pedido;
    2.3. A forma de entrega é exclusiva de delivery;
    2.4. A forma de pagamento é exclusiva de dinheiro;

3. O usuário pode fazer ações em relação à sua conta:
    3.1. Cadastrar: o usuário pode se cadastrar no sistema;
    3.2. Atualizar dados: O usuário pode atualizar seus dados no sistema;
    3.3. Deletar conta: O usuário pode deletar sua conta no sistema;
    3.4. Login: o usuário pode entrar (fazer login) no sistema;

Observações:
1. Convém ressaltar que, para inserir os produtos, é necessário adicioná-los ao banco de dados, pois eles só são mostrados dessa maneira na página de compra.
