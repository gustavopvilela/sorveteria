Título do projeto: Controle de sorveteria

Equipe:
- Eric Felipe Santos;
- Gustavo Henrique Pereira Vilela;
- Pedro Gabriel Teixeira de Oliveira.

Descrição das funcionalidades:

Do lado do cliente:
1. Poder escolher o que pedir;
2. Poder ver meus itens no carrinho;
3. Poder escolher modificações no meu pedido;
4. Poder escolher se é pra entregar ou retirar na sorveteria;
	4a. Se for pra entregar, poder escolher o endereço de entrega;
5. Poder escolher a forma de pagamento;
	5a. Se for dinheiro, poder escolher se precisa de troco ou não;
	5b. Se for cartão, pedir pra trazer a maquininha;
6. Poder ver o tempo de entrega;
7. Poder excluir itens do carrinho, se assim desejado;
8. Poder ver os pedidos anteriores;
9. Poder cadastrar o cliente;

Do lado do funcionário:
1. Poder ver os pedidos feitos pelos clientes;
2. Poder marcar o pedido do cliente, como visto e saída para entrega;
